# site-v2 — Shark-ISR VTOL portfolio site

Next.js 15 + Three.js/R3F engineering portfolio for the autonomous shark-monitoring VTOL project.
Showcases the autonomy architecture, hardware stack, perception pipeline, and SITL results.

## Dev

```bash
npm install
npm run dev        # http://localhost:3000
```

## Build

```bash
npm run build
npm start
```

See the [root README](../README.md) for full project context.
